# ADR-NNNN: Short title

- **Status:** proposed | accepted | superseded by ADR-XXXX | deprecated
- **Date:** YYYY-MM-DD
- **Deciders:** names

## Context

What problem is this ADR solving? What constraints exist? What facts are relevant? Keep it factual — opinions and tradeoffs go in the next section.

## Decision

What did we decide to do? State it crisply, in one or two sentences. Then expand if needed.

## Consequences

What does this enable? What does it foreclose? What new work does it create? What ongoing cost does it carry? Be honest about downsides — an ADR with no downsides usually means we haven't thought hard enough.

## Alternatives considered

Each alternative gets a short paragraph: what it was, why we didn't pick it. The point of this section is to make future-readers' lives easier when they're tempted to "just" change the decision.
